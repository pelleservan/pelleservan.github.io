Human Resources DashBaords Visualisation de dnooées

PrésentationRetour

Human Resources DashBoards est un projet de visualisation de date. L'idée est d'avoir une application web communiquant avec une base de données. Ainsi nous pouvons réaliser un certain nombre de graphiques dynamiques de différents types. En effet, étant donnée qu'a chaque fois que l'on charge une page de cette application on interroge la base de données, si cette dernière est mise à jour alors l'application l'est aussi.

Cette application est rattachée à la base de données humanresources. Celle-ci concerne les données relatives aux ressources humaines d'une compagnie. Elle est constituée de plusieurs tables et ce construit sur un modèle relationnel.

L'objectif de ce projet est de pouvoir visualiser de manière dynamique les données brutes tout en pouvant les filtrer mais aussi de construire des dashboards. À l'avenir l'application offrira la possibilité de fournir des reportings directement téléchargeables.
